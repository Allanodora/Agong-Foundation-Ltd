# Agong Foundation Ltd Website

Professional static V1 website for Agong Foundation Ltd.

## How to preview locally

Open `index.html` in a browser.

## Files

- `index.html` - main website
- `css/style.css` - styling
- `js/main.js` - mobile menu
- `assets/logo.png` - enhanced logo
- `assets/hero.jpg` - community photo
- `assets/favicon.png` - browser icon

## Before final launch, confirm

- Correct slogan: `Dako En Teko` vs `Dako En Pagi`
- Member names and titles
- Strategic plan details
- Activity/program details
- Donation/payment details
- Final domain name
